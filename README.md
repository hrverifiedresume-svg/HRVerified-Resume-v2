# HRVerified Resume v2

A comprehensive platform for resume building, job matching, interview preparation, and skill gap analysis.

## Features
- Resume builder with professional templates
- Job description matching with AI
- Interview question generation
- Skill gap analysis and recommendations
- Alumni networking and community
- Multi-role support (freshers, professionals, organizations)

## Tech Stack
- Frontend: React/TypeScript with Tailwind CSS
- Backend: Node.js/Express with TypeScript
- Database: Supabase (PostgreSQL)
- Authentication: Gmail, Magic Link, OTP, LinkedIn OAuth
- AI Integration: Claude API, Google Generative AI, Hugging Face
